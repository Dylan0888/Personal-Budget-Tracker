# Personal Budget Tracker (Still in development)

A responsive personal finance tracking web app built using React. Allows users to monitor income and expenses, categorise transactions, and visualise spending habits directly in the browser. The application demonstrates state-driven UI updates, form handling, and client-side persistent data.
